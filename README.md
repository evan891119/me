# me

First-person 3D portfolio museum, built as a desktop-first Vite + React + TypeScript + React Three Fiber app.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run typecheck
npm run lint
npm run build
```

## Preview

```bash
npm run preview
```
